# Camel Race Simulator 🐫

This program simulates a camel race where multiple camels compete to reach the finish line.  
Each camel moves forward randomly, and the user can place a bet on which camel will win.

The game includes random events such as speed boosts and stumbling to make the race more fun and unpredictable.

---

## Features

- Camel class to store name and distance
- Random movement system
- Speed boost and stumble events
- Betting system
- Ranking during the race
- Replay option

---

## How to Run

1. Run the program
2. Enter camel names
3. Choose a camel to bet on
4. Watch the race and see if you win!
